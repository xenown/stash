# Stash Backend

Reponsible for making the requests to Plaid to get your financial data and
storing this data in a local mysql file.


## Instructions for Running:
1. Go to Plaid.com and get your API secrets
2. Copy those into the .env file locally. Do not push any .env files online!
3. Launch on
